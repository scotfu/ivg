This project is still under developing. Features may change over time.

The main idea of this project is to serve as a general platform of dimensional reducation.
You could upload a csv file that contains numeric data with other categorical data. The system will take a few minutes to do precomputation on a few algorithms(MDS,PCA) and redirect you to the plot page where you can explore further more the data with a couple gadgets.

K-means will allow you to select a couple points as the centroids, then run sigle step k-means on the whole dataset. You could do this over again three times. Zoom-in could be achived in this way. The special 'margin' cluster is used to fix the scale of the plot
Enable Brush is a feature that allows you to select mutiple points, then generate a barchart to the right of the scatter plot where you could see the average values of each dimensions on selected points. You can click on any of the bars to get a histogram on that selected dimention. In order to do the K-means, you need to disable brush first.
